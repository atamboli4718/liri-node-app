#Overview
This app displays information based on user inputs for movies, songs, and concerts. The purpose of this app was to strengthen Nodejs skills, JavaScript, and API's.

#Technologies used
**JavaScript**
**Nodejs**
**BandsInTown API**
**Spotify API**
**OMDB API**
**Node Packages**
    - Axios
    - DOTEMV
    - Inquirer
    - Moment
    - Node-Spotify-api
